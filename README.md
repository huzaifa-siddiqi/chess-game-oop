# ♟️ Chess Game in C++ (OOP Project)

## 📌 Description

This is a console-based Chess Game implemented in C++ using Object-Oriented Programming concepts. The game allows two players to play chess by following standard rules including valid moves, check, checkmate, stalemate, castling, and pawn promotion.

---

## 🚀 Features

- 8×8 Chess Board Display
- Two Player Mode (White vs Black)
- Valid Move Checking
- Check & Checkmate Detection
- Stalemate Detection
- Castling Support
- Pawn Promotion
- Dynamic Memory Management

---

## 🧠 OOP Concepts Used

- Encapsulation
- Inheritance
- Polymorphism
- Composition

---

## 🛠️ How to Run

### Compile:

```bash
g++ main.cpp -o chess
```

### Run:

```bash
./chess
```

---

## 🎮 How to Play

- Input format:

```
Current position: e2
Moving position: e4
```

- Columns: a–h
- Rows: 1–8
